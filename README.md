# ストップウォッチ

HTML、CSS、JavaScript だけで動作するシンプルなストップウォッチです。

## 機能

- 開始
- 停止
- リセット
- ラップタイム記録
- スマホ対応のレスポンシブデザイン

## ファイル構成

- `index.html` - アプリ本体の HTML
- `style.css` - デザインとレイアウト
- `script.js` - ストップウォッチの動作ロジック

## 使い方

1. `index.html` をブラウザで開く
2. `開始` ボタンを押して計測を開始する
3. `停止` で一時停止
4. `ラップ` で現在の経過時間を記録
5. `リセット` で時間を 0 に戻し、ラップを消去

## 補足

- `style.css` で背景色やレイアウトの調整ができます。
- `script.js` でタイマーの更新間隔やラップ挙動を変更できます。
